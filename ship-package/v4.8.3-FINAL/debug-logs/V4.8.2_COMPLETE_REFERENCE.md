## Guardian Enforcement (v4.8.2)

### BRAIN_FAILURE_BLOCK
When brain is unknown (uninitialized), dangerous tools are HARD BLOCKED:
```
[BRAIN_FAILURE_BLOCK] Cannot execute dangerous tool "terminal" when system brain is uninitialized.
```

### SUDO SYSTEM
Config edits require approval via SHARK_SUDO_TOKEN environment variable.

### CONTAINER_LEAK_BLOCK
Deployment blocked if containers are running.
